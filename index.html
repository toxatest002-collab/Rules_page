ПРАВИЛА СЕРВИСА

Настоящие Правила определяют условия использования онлайн-сервиса (далее — «Сервис») и являются публичной офертой в соответствии с законодательством Российской Федерации.

Нажатие кнопки «Я согласен с правилами сервиса, начать тест» означает полное и безоговорочное принятие Пользователем настоящих Правил.

1. Общие положения

1.1. Сервис представляет собой онлайн-инструмент (тест, опрос, игровой, развлекательный или информационный сервис), позволяющий Пользователю проходить тестирование и получать результат на основании предоставленных ответов.

1.2. Использование Сервиса допускается только при полном согласии Пользователя с настоящими Правилами.

2. Порядок использования

2.1. Перед началом использования Сервиса Пользователь обязан ознакомиться с настоящими Правилами.

2.2. Нажатие кнопки «Начать тест» означает подтверждение ознакомления с Правилами, предоставление согласия на обработку персональных данных, а также согласие на передачу данных третьим лицам в объёме, предусмотренном Сервисом.

3. Обработка данных

3.1. В рамках использования Сервиса могут обрабатываться следующие данные: ответы Пользователя в тесте, результаты тестирования, технические данные (ID пользователя, First Name, Last Name и другие сопутствующие данные), а также данные о действиях Пользователя в Сервисе.

3.2. Пользователь подтверждает, что предоставляет данные добровольно. Данные обрабатываются для целей предоставления функционала Сервиса, формирования и отображения результата теста, улучшения качества Сервиса, распространения информации, проведения статистических и аналитических исследований, а также разработки новых функций и алгоритмов.

Сервис вправе осуществлять любые действия с данными, включая сбор, запись, систематизацию, накопление, хранение, уточнение, использование, передачу, блокирование и удаление.

4. Изменение правил

4.1. Сервис вправе изменять настоящие Правила.

4.2. Новая редакция вступает в силу с момента её публикации. Продолжение использования Сервиса означает согласие с новой редакцией.
