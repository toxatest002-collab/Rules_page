ПРАВИЛА СЕРВИСА

Настоящие Правила определяют условия использования онлайн-сервиса (далее — «Сервис») и являются публичной офертой в соответствии с законодательством Российской Федерации.

Нажатие кнопки «Начать тест» означает полное и безоговорочное принятие Пользователем настоящих Правил.

1. Общие положения

1.1. Сервис представляет собой онлайн-инструмент (тест/опрос/игровой, развлекательный или информационный сервис), позволяющий Пользователю проходить тестирование и получать результат на основании предоставленных ответов.

1.2. Использование Сервиса допускается только при полном согласии Пользователя с настоящими Правилами.

2. Принятие условий

2.1. Перед началом использования Сервиса Пользователь обязан ознакомиться с настоящими Правилами.

2.2. Нажатие кнопки «Начать тест» означает: подтверждение ознакомления с Правилами; предоставление согласия на обработку персональных данных; согласие на передачу данных третьим лицам в объёме, предусмотренном Сервисом.

3. Персональные данные и данные Пользователя

3.1. В рамках использования Сервиса могут обрабатываться следующие данные: ответы Пользователя в тесте; результаты тестирования; технические данные (ID пользователя, First Name, Last Name и другие сопутствующие данные); данные о действиях Пользователя в Сервисе.

3.2. Пользователь подтверждает, что предоставляет данные добровольно.

4. Цели обработки данных

Данные обрабатываются для следующих целей: предоставление функционала Сервиса; формирование и отображение результата теста; улучшение качества Сервиса; распространение информации; проведение статистических и аналитических исследований; разработка новых функций и алгоритмов.

5. Обработка и хранение данных

Сервис вправе осуществлять любые действия с данными, включая: сбор; запись; систематизацию; накопление; хранение; уточнение; использование; передачу; блокирование; удаление.

6. Изменение Правил

6.1. Сервис вправе изменять настоящие Правила.

6.2. Новая редакция вступает в силу с момента её публикации. Продолжение использования Сервиса означает согласие с новой редакцией.
